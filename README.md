# Global Sales Analysis - Anudip Data Visualization Training

This repository contains the project files for the **Global Sales Analysis (Major Project)**, completed as part of the Anudip Data Visualization training program. The project focuses on cleaning, exploring, and visualizing a large-scale global sales dataset to uncover key business insights regarding revenue, profit margins, item types, and regional performance.

## 📂 Project Structure

- **`data/`**: Contains the dataset used for analysis.
  - `raw_global_sales_data.csv`: The initial, unprocessed dataset.
  - `cleaned_global_sales_data.csv`: The processed dataset after data cleaning, handling missing values, and formatting.
- **`notebook/`**: Contains the Jupyter Notebook with the Python code for data analysis and visualization.
  - `global_sales_analysis (Major Project).ipynb`: The main notebook covering data cleaning, Exploratory Data Analysis (EDA), and graphical representations.
- **`visual graphics/`**: Contains 24 high-quality exported visualizations generated during the analysis. Some examples include:
  - Revenue and Profit by Region & Country
  - Item Type Margin and Profitability Analysis
  - Yearly and Monthly Sales Trends
  - Correlation Heatmaps and Margin Distributions

## 📊 Key Analysis Areas

The project explores various facets of the global sales data, including but not limited to:
- **Geographic Performance**: Identifying top and bottom performing countries and regions based on revenue and profit.
- **Product Profitability**: Analyzing which item types yield the highest margins and total profit.
- **Time-Series Trends**: Evaluating sales performance across different years, quarters, months, and days of the week.
- **Sales Channels & Order Priorities**: Understanding the impact of sales channels (Online vs. Offline) and order priorities on profitability and shipping times.

## 🛠️ Technologies & Libraries Used

- **Python**: Primary programming language for data analysis.
- **Pandas & NumPy**: For data manipulation, cleaning, and numerical computations.
- **Matplotlib & Seaborn**: For creating static, insightful data visualizations.
- **Jupyter Notebook**: For interactive development and presenting the analysis workflow.

## 🚀 How to Run

1. Clone this repository:
   ```bash
   git clone https://github.com/shishirastogi/anudip_project.git
   ```
2. Navigate to the project directory:
   ```bash
   cd anudip_project
   ```
3. Open the Jupyter Notebook:
   ```bash
   jupyter notebook "notebook/global_sales_analysis (Major Project).ipynb"
   ```
   *(Ensure you have Jupyter and the required Python libraries like pandas, matplotlib, and seaborn installed in your environment).*
